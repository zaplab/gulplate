gulplate
=========

Basic gulp Template

Mirrors https://github.com/zaplab/gruntlate tasks.

## Setup

```
$ npm install
$ bower install
$ gulp setup
```

## Usage

```
$ gulp
$ gulp serve
```

### If we take use of isDevMode inside gulpfile.js

development (default) or production output (for example css compression on/off):

```
$ gulp --target=production
$ gulp --target=prod
```
